# 🌱 GOTTA – Sistema de Monitoramento Ambiental com Drones

Projeto acadêmico que simula uma empresa de drones voltada ao monitoramento ambiental e prevenção de queimadas, com foco na coleta, análise e visualização de dados.

## 📌 Sobre o projeto

O **GOTTA** é uma aplicação web desenvolvida com o objetivo de analisar dados ambientais e gerar diagnósticos automatizados sobre possíveis riscos de queimadas.

O sistema permite o cadastro de informações relevantes, processa esses dados e fornece relatórios e alertas que auxiliam na tomada de decisão, simulando um cenário real de uso tecnológico para preservação ambiental.


## 🚀 Funcionalidades

* 📊 Cadastro e gerenciamento de dados ambientais
* 🔍 Análise de informações para identificação de riscos
* 📄 Geração automática de relatórios de diagnóstico
* ⚠️ Sistema de alertas para possíveis queimadas
* 🗂️ Organização e armazenamento de dados


## 🛠️ Tecnologias utilizadas

* **HTML** – Estrutura da aplicação
* **CSS** – Estilização e layout
* **PHP** – Lógica de back-end
* **MySQL** – Banco de dados


## 🎯 Objetivo

Aplicar na prática conceitos de desenvolvimento web e análise de dados em um projeto com contexto realista, utilizando tecnologia como ferramenta para solução de problemas ambientais.
